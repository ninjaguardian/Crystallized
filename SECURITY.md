# Security Policy

## Supported Versions


| Minecraft Version | Supported          |
| ------- | ------------------ |
| 1.20.1   | :white_check_mark: |
| < 1.20.1   | :x:                |

## Reporting a Vulnerability

If there is a Vulnerability, please put it in the issues tab as 'bug' and I will try and fix it as soon as I can.
